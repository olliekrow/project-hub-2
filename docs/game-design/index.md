# Game Design Docs
{: .no_toc }

GDD templates and reference documentation for mobile and kids' game projects.

---

## Standards

All GDDs follow these conventions:

- **Personas** use card-style blocks (not side-by-side tables) for readability
- **Language** is gender-neutral throughout — descriptive names and they/them pronouns
- **Format** targets Google Docs for collaborative editing, with this wiki as the reference index
